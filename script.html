JavaScript Basics

This section explains the basic concepts of JavaScript.

JavaScript has 7 basic data types

  • string

    A string is a sequence of characters, like this sentence.

  • number

    A number can be an integer like 42 or a decimal like 3.14.

  • boolean

    • A boolean is either true or false.

  • bigint

    • A bigint represents very large numbers, like 9007199254740991n

  • symbol

    • A symbol is a unique and immutable identifier: Symbol('example')

  • null

    • Null represents the intentional absence of any object value.

  • undefined

    • Undefined means a variable has been declared but not assigned a value.

There are two main categories of data types in JavaScript: primitive and non-primitive (or reference). Primitive types include the string data type, which holds text, the number type for integers or floating-point values, boolean for true or false values, null to indicate a special kind of absence of value, undefined to show that a variable has not yet been assigned, and symbol, which is used for creating unique identifiers. Non-primitive types, such as objects, are collections of properties and methods that can hold multiple values or more complex data structures. JavaScript is dynamically typed, meaning that the data type of a variable is determined at runtime and can change as the program executes, providing flexibility but also necessitating careful management of types to prevent errors.
